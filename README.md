# bacteria_cup_trainer
Интерактивный тренажёр задачи про бактерии в стакане: наглядная модель экспоненциального роста, где количество удваивается каждый шаг. Позволяет задавать начальное число бактерий, вместимость стакана и отслеживать заполнение по шагам. Помогает быстро понять ключевую идею задачи — от половины до полного объёма всего за один шаг.
